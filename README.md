## 🧩 Sobre o Projeto

O ScaleMind é uma aplicação web que gera um guia nutricional completo e personalizado. O usuário informa peso, altura, idade, sexo e nível de atividade — o app calcula o metabolismo basal pela **Fórmula de Mifflin-St Jeor**, aplica o fator de atividade e retorna resultados detalhados.

---

## 🚀 Funcionalidades

- 🔥 Meta calórica diária personalizada por objetivo (emagrecer, manter, ganhar massa)
- 🥩 Distribuição de macronutrientes em gramas (proteína, carboidrato e gordura)
- 💧 Recomendação de ingestão de água e fibra por sexo biológico
- 🏋️ Sugestão de atividade física baseada no objetivo do usuário
- 🧭 Navegação por estado entre 3 telas com steps dinâmicos
- ✅ Validação de campos com shake animation nos inputs obrigatórios
- 🔢 Animação de contador com `requestAnimationFrame` e easing suave
- 📱 Layout responsivo testado de 320px ao desktop

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3 (Glassmorphism, orbs animados, paleta Ember & Moss, Flexbox, Grid)
- JavaScript Vanilla (requestAnimationFrame, navegação por estado, validação, DOM Manipulation)
