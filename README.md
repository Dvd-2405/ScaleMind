# ScaleMind---Guia-Nutricional 
Acabei de lançar o ScaleMind — uma calculadora nutricional que desenvolvi do zero com HTML, CSS e JavaScript puro. 🧠⚖️

A ideia é simples: o usuário informa peso, altura, idade, sexo e nível de atividade. O app calcula o metabolismo basal pela Fórmula de Mifflin-St Jeor, aplica o fator de atividade e gera um guia nutricional completo com:

→ Meta calórica diária personalizada
→ Distribuição de macros (proteína, carbo e gordura em gramas)
→ Recomendação de água e fibra por sexo biológico
→ Sugestão de atividade física por objetivo

Além da lógica de cálculo, me desafiei também no front:

✦ Animação de contador com requestAnimationFrame e easing suave
✦ Validação com shake animation nos campos obrigatórios
✦ Navegação por estado entre 3 telas com steps dinâmicos
✦ Design responsivo testado de 320px ao desktop
✦ Paleta Ember & Moss com glassmorphism e orbs animados

Cada projeto que entrego me faz entender melhor que front-end não é só código — é experiência.
